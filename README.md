A weboldal témája egy online webshop. Cipők, nadrágok és pólók vásárlása lehetséges az oldalon. Az inspo.html oldalon még inspirációt is lehet venni modellekről.
SCRIPTEK
    adatbetoltes.js
        A script a főoldalon (home.html, index.html) lévő termékek betöltésére szolgál. Egy listából betölti a html fájlba egyesével a termékeket
    A script.js fileban két kód van, az oldalon lévő két formhoz.
        Az első script a kapcsolat.html oldalon lévő form validálására szolgál
        A második script pedig a főoldalon (home.html, index.html) lévő filter használatára
            A filterrel lehet választani a ruha típusok, árak, illetve akciós termékek közül. És azután az oldal helyesen leszűr a feltételeknek megfelelően.
    mix.js
        A script a mix.html oldalon lévő program működésére szolgál
            Az "új szett" gomb lenyomásával mindig egy random ruha összeállítást kap a felhasználó a listában megadott ruhákból, egy cipő, egy nadrág, illetve egy pólót ad a rendszer
            A kapott ruhadaraboknak az árát összeadja és kiírja alulra
Használt font
    A Bebas Neue fontot használtam minden oldalon
    A font minden oldal head részében linkelve van
    "https://fonts.googleapis.com/css2?family=Bebas+Neue&display=swap"
Külső források
    Az oldal elkészítéséhez inspiráció érdekében használtam ChatGpt-t illetve Gemini Prot
    Illetve az oldal alapja egy általam készített régebbi webshop, ami javascript nélkül készült el, de teljesen reszponzív volt, így a css és html szinte teljesen saját munka
    A script.js és mix.js-t eredetileg ai csinálta, de szinte az egészet újra írtam, kijavítottam saját gondolatokkal
